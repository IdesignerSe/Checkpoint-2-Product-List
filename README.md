## Checkpoint 2 – Product List Management System ##
 
Create a Console Application in C# where users can manage a product list. 

The application should allow users to: 
• Add products  ✅
• View products  ✅
• Search products  ✅
• Calculate total price  ✅
• Sort products  ✅
• Handle invalid input ✅ 
• Continue managing products dynamically  ✅

The application should focus on: 
• Object-Oriented Programming (OOP)   
• Classes & Methods  ✅
• Lists & Collections  ✅
• LINQ  ✅
• Error Handling  ✅
• Console UI Design ✅  
• Real-world coding structure ✅ 

User Input 
The application starts by asking the user for: 
1. Product Category   ✅
2. Product Name       ✅
3. Product Price      ✅

Bonus 
1) Sorting (ascending/descending) ✅
Easy, fast, and improves usability immediately.

2) Filtering by price range  ✅
Pairs perfectly with sorting.

3) Pagination ❌
Makes your ShowProducts() look professional.

4) Category colors ❌
Cosmetic but fun and improves readability.

5) Export report to TXT ❌
Uses your existing ShowProducts() logic.

6) Admin/User mode ❌
Adds authentication + permissions.

7) Generic repository structure ❌
Refactors your code into a professional architecture.

8) Unit testing ❌
Final polish — tests for Add/Edit/Delete/Load/Save.
